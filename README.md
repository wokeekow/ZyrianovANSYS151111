1. Знакомство с продуктовым продуктом HFSS.
## 1.1. Методы расчета антенн в свободном пространстве:
Метод конечных элементов (Finite Element) - в объеме, в пространстве. Задается граничное условие, которая ограничивает область анализа. Позволяет очень точно и качественно промоделировать электромагнитные поля в структуре. Достает все ЭМ характеристики устройства в ближних и дальних зонах. 3D. Требует достаточное количество ОЗУ.
Метод интегральных уравнений (Integral Equation) - на поверхности геометрии, хорошо подходит для больших металлических структур. Не требуется задача граничного условия. Позволяет получать диаграммы направленности в дальней зоне, но в основном этим методом рассматриваются металлические структуры. Создается 2D сетка на поверхностях анализа. Требует меньшее количество ОЗУ, по сравнению с FE.
Гибридные регионы FE-BI – комбинация FE + IE.  Облегчает совместное моделирование на расстоянии. Пример: маленькая структура (облучатель) рассчитывается с граничным условием, а большая структура рассчитывается интегральным методом в 2D сетке по плоскости. Умеренное количество ОЗУ в больших задачах.
SBR+ - метод стреляющих и отраженных лучей FE + SBR. Преодолевает расстояние без необходимости построения сетки в объеме. Использует графические ускорители для расчета.

Тут другой принцип: сетку строят не внутри всего объёма, а только на поверхности металла. Для больших железных конструкций — самое то. Памяти требует меньше. И информацию про поля в ближней зоне даёт не так детально, зато диаграммы направленности в дальней зоне — пожалуйста.

# ЧТО ТАКОЕ ЭПР
ЭПР (Эффективная Площадь Рассеяния) или RCS (Radar Cross Section) — это физическая величина, которая характеризует заметность объекта для радара.
Простыми словами: ЭПР — это размер «воображаемой» металлической сферы, которая отражала бы сигнал так же, как этот объект.
Если объект (истребитель, ракета) имеет ЭПР 1 м², это значит, что на экране радара он выглядит как идеально проводящий шар площадью 1 м², даже если физически он размером с многоэтажный дом. 
ЭПР зависит от:
Размера: Крупный объект (корабль, бомбардировщик) обычно заметнее.
Материала: Металл отражает отлично, дерево или пластик — хуже, радиопоглощающие материалы (РПМ) — гасят сигнал.
Формы: Плоские грани (как у «стелс») уводят сигнал в сторону, не возвращая его к радару. А прямые углы (уголковые отражатели) создают колоссальную ЭПР.
Ракурса: Сбоку самолет может быть огромным пятном, спереди — почти невидимым.
Длины волны: Если длина волны радара сравнима с размерами объекта, возникает резонанс, и ЭПР скачкообразно растет.
